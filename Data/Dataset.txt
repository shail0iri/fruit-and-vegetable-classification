<📊 Dataset
Dataset Information
Total Size: ~3.1 GB
Number of Classes: 36 different fruits and vegetables
Image Format: JPEG
Resolution: Various sizes (resized to 224×224 for training)

📥 How to Get the Dataset
Download from Original Source

The dataset is too large for GitHub. Please download from: [Dataset](https://www.kaggle.com/code/abdelrahman16/fruit-and-vegetable-classification/input)

Manual Setup
Download the dataset from the original source
Extract to the data/ folder

🗂️ Dataset Statistics
Split -	Number of Images-	Size
Train	-15,000 images	-2.3 GB
Test	-3,000 images	-450 MB
Validation	-1,500 images	-225 MB
Total	-19,500 images	-3.1 GB

📋 Classes Included
The dataset contains 36 classes including:
Fruits: Apple, Banana, Orange, Grapes, Mango, Strawberry, Pineapple, etc.
Vegetables: Tomato, Cucumber, Carrot, Bell Pepper, Onion, Potato, Broccoli, etc.
